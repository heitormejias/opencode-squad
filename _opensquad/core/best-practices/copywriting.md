# Copywriting Best Practices

## Principles

1. **Clarity First**
   - Simple words, short sentences
   - One idea per paragraph
   - Active voice

2. **Benefit-Driven**
   - Lead with benefits, not features
   - Show transformation
   - Address pain points

3. **Action-Oriented**
   - Clear CTAs
   - Urgency when appropriate
   - Easy next steps

## Structure

### Headline
- Benefit-focused
- Specific numbers
- Emotional hook

### Body
- Problem → Agitation → Solution
- Social proof
- Objection handling

### CTA
- Specific action
- Low friction
- Clear value

## Format

```markdown
# [Benefit-Driven Headline]

[Hook/Opening that addresses pain point]

## The Problem
[What they're struggling with]

## Our Solution
[How you solve it]

## Key Benefits
1. [Benefit 1]
2. [Benefit 2]
3. [Benefit 3]

## Proof
[Testimonial/Social proof]

## Call to Action
[Specific next step]
```

## Quality Criteria

- [ ] Headline captures attention
- [ ] Clear value proposition
- [ ] Addresses objections
- [ ] Strong CTA
- [ ] Appropriate tone for audience
