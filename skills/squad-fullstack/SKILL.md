# Squad Fullstack

Squad completo para desenvolvimento de aplicações fullstack modernas.

## Pipeline de Execução

```
┌─────────┐    ┌────────────┐    ┌────────────┐    ┌───────────┐    ┌────────────┐
│   UX    │───▶│  FRONTEND   │───▶│   BACKEND   │───▶│   MOBILE   │───▶│ SEGURANÇA  │
│Design   │    │   Web       │    │   API       │    │   App      │    │   Audit    │
└─────────┘    └────────────┘    └────────────┘    └───────────┘    └────────────┘
```

## Agentes

1. **UX Designer** - Pesquisa, wireframes, protótipos
2. **Frontend Dev** - Web app (React/Next.js)
3. **Backend Dev** - APIs, microsserviços, DB
4. **Mobile Dev** - Apps iOS/Android (React Native/Flutter)
5. **Security Auditor** - Code review, penetration testing

## Skills do Squad

| Agente | Skills |
|--------|--------|
| UX Designer | ux |
| Frontend Dev | dev-frontend, ux |
| Backend Dev | dev-backend |
| Mobile Dev | dev-mobile |
| Security Auditor | auditor-seguranca |

## Output

- Design System
- Web App (Next.js/React)
- API (Node.js/Python/Go)
- Mobile App (React Native)
- Security Report
